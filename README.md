# Shopping Cart UI

Simple shopping cart app.

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 8.1.2.

## Development server

Run `npm i` to install dependencies.

Run `npm start` for a dev server. Navigate to `http://localhost:4200/shopping-cart`. The app will automatically reload if you change any of the source files.

## Set up Backing API
Follow instructions here for setting up API: https://github.com/yenni-tan/shopping-cart-api
